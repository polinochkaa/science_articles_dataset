Непредсказуемое нерегулярное поведение детерминированных нелинейных 
динамических систем в последние несколько десятилетий стало очень интересным 
предметом для исследования. Большое внимание привлекают системы, способные 
демонстрировать хаотическое поведение с несколькими степенями свободы, изме-
нение управляющих параметров в которых   сопровождается различными бифурка-
циями. Для таких систем целесообразно исследовать, как трансформируется хаоти-
ческий режим с изменением значений параметров или с увеличением числа степе-
ней свободы.Чтобы решить подобную задачу, требуются количественные характе-
ристики свойств хаотического поведения. Удобными и надежными такими харак-
теристиками хаоса являются экспоненциальная дивергенция ближайших орбит (по-
ложительные показатели Ляпунова), положительная конечная энтропия Колмого-
рова и нецелая размерность аттрактора. Между этими величинами существует не-
сколько соотношений, и, если определить спектр Ляпунова, остальные можно оце-
нить через равенства или сделать для них оценки сверху или снизу. 
С необходимостью анализа хаотических режимов зачастую сталкиваются экс-
периментаторы. Поэтому исследование и разработка методов расчета показателей 
Ляпунова ДЛЯ нерегулярных временных рядов, иногда всего лишь скалярных, 
представляет немалый интерес. 
В данной работе сравниваются показатели Ляпунова, полученные при помощи 
различных методов - по уравнениям в вариациях (спектр показателей), методом 
Вольфа [1] (только наибольший показатель), методом Экманна [2] (спектр показа-
телей) и методом Сано-Савада (спектр показателей) [3]. 
Последний метод имеет более высокую точность для относительно коротких 
временных рядов [4-6]. Обладая большей статистической значимостью уже для ло-
кальных оценок показателей (т.е. для линейного оператора эволюции возмущения), 
он более эффективен по сравнению с адаптированным для временных рядов алго-
ритмом Бенеттина. Кроме того, в силу непосредственного вычисления линейного 
оператора эволюции метод может  оказаться удобным для расчета, например, ко-
вариантных ляпуновских векторов [7], для применения алгоритма проверки струк-
турной устойчивости (грубости) хаотического режима [8], для выявления паразит-
ных показателей [9].
